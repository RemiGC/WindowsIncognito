Windowsincognito

A small software that create a simplifed Incognito Mode for Windows Explorer

View on GitHub Download .zip Download .tar.gz

Welcome to Windows Incognito

This small program can lock the files that store the last uses files creating an incognito like feature for Windows Explorer. While the program is on and all files are locked, all the files you open, the folders you browse etc... won't appear in quick access in Windows 10.

More Details

The program open all files in %APPDATA%\Microsoft\Windows\Recent\AutomaticDestinations in write mode preventing other applications from updating them.
